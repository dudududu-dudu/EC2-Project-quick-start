日本での生活・ビジネスを法律の専門家がサポートします

不動産登記、会社設立、帰化申請など中国語で安心サポート。専門家にお任せください。

業務案内

1．不動産登記

外国人の方が日本で不動産を購入・売却・相続される際の登記手続きをお任せください。契約書作成から申請まで丁寧にサポート。

2．会社・法人設立

株式会社設立、外国会社支店・駐在員事務所設立などを分かりやすくご案内。有限会社→株式会社移行登記も対応。

3．会社・商業登記

商号変更、本店移転、資本金増資、解散清算など法人登記事項を幅広く支援。

4．相続登記・遺産承継

名義変更、遺産分割協議書作成、法定相続情報一覧図取得など複雑な相続手続きをトータルでサポート。

5．成年後見制度支援

後見申立書類の作成や家庭裁判所手続きなど、制度利用を専門家が支援します。

6．遺言書作成

自筆証書遺言・公正証書遺言の原案作成および公証立会いを全国対応。

7．帰化申請・国籍取得

書類準備から面接対策まで、日本国籍取得をワンストップサポート。

8．各種許認可申請

飲食店営業許可・古物商許可・建設業許可など、事業開始に必要な許認可取得を代行。

9．裁判所提出書類作成

相続放棄申述書、特別代理人選任申立書等、裁判所提出書類の作成を言語の壁なく支援。

10．その他生活トラブル相談

契約トラブル、借金問題など日常の法律相談もお気軽に。
