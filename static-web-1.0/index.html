日本での生活・ビジネスを法律の専門家がサポートします

不動産登記、会社設立、帰化申請など中国語で安心サポート。専門家にお任せください。

業務案内

1．不動産登記

外国人の方が日本で不動産を購入・売却・相続される際の登記手続きをお任せください。立会業務にも対応可能です。贈与契約書・売買契約書の作成から、登記申請まで丁寧にサポートいたします。

2．会社・法人設立

日本で会社を設立する外国人の方もサポートします。株式会社設立、外国会社の日本支店・駐在員事務所設立、法人登記手続きまで、分かりやすくご案内します。有限会社から株式会社への移行登記など、特殊なケースにも対応可能です。煩雑な手続きも司法書士にお任せいただければ、正確かつ迅速に対応いたします。

3．会社・商業登記

会社名（商号）変更、本店移転、資本金の増資、解散・清算結了など、法人に関わる登記事項の手続きを幅広くサポートいたします。大阪を拠点としつつ、全国対応も承っておりますので、遠方の方もお気軽にご相談ください。

4．相続登記・遺産承継手続きサポート

外国籍の方が日本国内の不動産を相続する場合、相続登記手続きが必要です。相続による不動産名義変更、遺産分割協議書の作成、法定相続情報一覧図の取得など、複雑な相続手続きをトータルでサポートいたします。外国籍であっても、出生証明書や婚姻証明書、宣誓供述書など、必要書類の準備から登記申請まで、丁寧にサポートいたします。円満な相続を実現するためのご相談ください。

5．成年後見制度の利用支援

日本に居住する外国人の方も、成年後見制度を利用することが可能です。認知症や精神的な障がいにより判断能力が低下した場合、ご本人やご家族の財産管理や契約行為を保護するための制度です。家庭裁判所への申立てや必要書類の準備、後見人の選任手続きなど、専門的なサポートを提供いたします。

6．遺言書作成

自筆証書遺言原案作成、公正証書遺言その他の遺言原案作成および公証の立会いなど全般的にサポートいたします。中国人の方の遺言作成にも対応します。(大阪以外も対応可）

7．帰化申請・国籍取得手続き

日本国籍を取得するための帰化申請手続きを、書類作成から面接対策までトータルサポートいたします。煩雑な手続きを分かりやすくご説明し、確実にサポートします。

8．各種許認可申請

外国人の方が日本で事業を行う際に必要な、飲食店営業許可・古物商許可・建設業許可など、各種許認可の取得手続きを代行します。日本でのビジネススタートを全面的に支援いたします。

9．裁判所提出書類作成業務

相続放棄申述書の作成、特別代理人選任の申立など、日本での生活やビジネスにおいて、裁判所への書類提出が必要となる場合がございます。言語の壁を感じることなく、安心してご相談いただけます。

10．その他生活トラブル相談

契約トラブル、借金問題、日常生活の法律相談も気軽にご相談ください。
